# Sorting Algorithms in Multiple Languages

## Description

This project provides implementations of classic sorting algorithms in various programming languages. It serves as a practical example for understanding and comparing these fundamental algorithms across different language paradigms.

## Installation

No external package dependencies are required for this project. You only need the runtime environments for the respective languages installed on your system.

## Usage

To run any of the sorting algorithm implementations, navigate to the project's root directory and use the following commands:

**JavaScript (Bubble Sort)**
```bash
node src/bubbleSort.js
```

**Python (Insertion Sort)**
```bash
python3 src/insertion_sort.py
```

**Java (Selection Sort)**
First, compile the Java file:
```bash
javac src/SelectionSort.java
```
Then, run the compiled class:
```bash
java -cp src SelectionSort
```

## Features

- **Bubble Sort** implemented in JavaScript.
- **Insertion Sort** implemented in Python.
- **Selection Sort** implemented in Java.

## Requirements

Ensure you have the following installed on your system:
- [Node.js](https://nodejs.org/)
- [Python 3](https://www.python.org/)
- [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/)
